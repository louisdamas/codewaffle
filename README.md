# codewaffle

Ce code permet de faire évoluer le robot  TurtleBot 3 WafflePI dans un mouvement circulaire, puis d'activer le capteur LIDAR dans une détection front. Enfin, le capteur LIDAR est utilisé afin de pouvoir repérer un objet à 360° autour du robot, et de déclencher l'arrêt du robot.
